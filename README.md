# Sobre o Projeto

Este repositório contém o **projeto final do curso "Introdução à Python para Ciência de Dados (40h)"**, realizado no contexto do projeto **Aranouá**.

O objetivo do projeto é consolidar os principais fundamentos de Python aplicados à análise de dados, incluindo manipulação, exploração e interpretação de informações.

Durante o desenvolvimento, foram praticados conceitos essenciais para a área de Ciência de Dados, com foco em lógica e estruturas de dados.

---

# Objetivos de Aprendizagem

- Compreender os fundamentos da linguagem Python
- Trabalhar com estruturas básicas de dados
- Aplicar lógica de programação em problemas simples
- Realizar operações básicas de manipulação de dados
- Desenvolver autonomia para iniciar estudos em Ciência de Dados

---

# Conteúdos Abordados

- Variáveis e tipos de dados
- Estruturas condicionais (`if`, `else`)
- Estruturas de repetição (`for`, `while`)
- Listas, tuplas e dicionários
- Entrada e saída de dados
- Manipulação básica de dados
- Lógica de programação aplicada
